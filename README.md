# MEject
The MEject Microsoft Windows Application  ejects and closes the first CD Tray.  This 
application is like the eject command in linux.

#Example Usage 1:  Open the CD Tray

```
C:\work> MEject
```

#Example Usage 2:  Close the CD Tray

```
C:\work> MEject -t
```
